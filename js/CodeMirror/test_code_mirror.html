# Principle The principle is to consider the trajectory as described by a complex number. For the speed we have: $v=\rho e^{i\omega t}$ ## Without excitation the movement is described by $$\dfrac{df}{dt}=i a f$$ The solution is $f=\rho e^{ia t}$ This can be found with distribution. Indeed from above we have in Fourier space: $$ \omega F = ia F $$ so $$ F = \rho \delta(\omega-ia) $$ and the solution is $$f = \rho e^{ia t}$$ ## With excitation the movement is described by $$\dfrac{dv}{dt}=i a v + e^{ib t} $$ But $TF(e^{ib t}) = \delta(b+\omega)$ So $$F = \delta(ib-\omega) /(ia-\omega)$$ which is in fact $$F = \delta(ib-\omega) /i(a-b)$$ hence a particular solution is $$f = e^{ib t}/(a-b)$$ Giving the general solution $$f = e^{ia t} + e^{ib t}/(a-b)$$ # Using speed only Using a complex notation for the speed of an ion ie , the classical cyclotronic differential equation with an harmonic excitation for describing the ions trajectory is : $$\dfrac{dv}{dt}=i \dfrac{qB}{m}v+ k e^{i \omega_{excit} t}$$ This can be rewritten $$\dfrac{dv}{dt}=i a v + k e^{i b t}$$ where $a=\dfrac{qB}{m}$ and $b=\omega_{excit}$ Expressing $v$ as a linear combination of $e^{i b t}$ and $e^{i a t}$, we find: $$ v= e^{i a t} + k \dfrac{e^{i b_i t}}{b-a}$$ With $$ k = (v_0-1)(b-a)$$ # Todo Differential equation with Fourier Transform.
